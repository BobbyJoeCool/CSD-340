Intro to CSS

In this class, we will be learning about CSS, or Cascading Style Sheets, which is the language used to control the visual presentation of content written in markup languages such as HTML and Markdown. While markup languages define the structure and meaning of content, CSS works alongside them to determine how that content is displayed to the user. CSS allows developers to create reusable style rules, often referred to as classes, that can be applied to multiple elements within a document to ensure consistency and simplify maintenance. For example, HTML and Markdown both support headers as a way to organize content, with HTML using tags such as <h1> </h1> and Markdown using the # symbol. With CSS, we can define exactly how each of those headers will appear on a page, including properties such as font family, font size, color, spacing, and whether text appears bold, underlined, or italicized, giving developers precise control over the look and feel of a website.
